![Capa do Projeto](https://portalwebdesigner.com/wp-content/uploads/2016/06/Exemplo-CSS-e-HTML.png)

# Sobre o Projeto

Este projeto aborda a criação de um rodapé de página web, contendo links que permitem o usuário a saber mais sobre a empresa Design Daily, escolher a linguagem em que deseja vizualizar o site e links diretos que encaminham o usuário até as redes sociais da empresa de designer proprietária do website em questão.
